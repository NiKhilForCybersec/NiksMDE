Installation Methods

Installation Commands 
RHEL/CENTOS (yum):
# Add Microsoft repo
sudo yum-config-manager --add-repo=https://packages.microsoft.com/config/rhel/8/prod.repo

# Install
sudo yum install mdatp

UBUNTU/DEBIAN (apt):
# Add Microsoft repo
curl -o microsoft.list https://packages.microsoft.com/config/ubuntu/22.04/prod.list
sudo mv ./microsoft.list /etc/apt/sources.list.d/

# Import key
curl https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > microsoft.gpg
sudo mv microsoft.gpg /etc/apt/trusted.gpg.d/

# Install
sudo apt update
sudo apt install mdatp

SUSE (zypper):
sudo zypper addrepo https://packages.microsoft.com/config/sles/15/prod.repo
sudo zypper install mdatp
