Prerequisites Checklist

Prerequisites
PREREQUISITES CHECKLIST:

LICENSING:
[ ] MDE P1/P2 or M365 E5 licenses procured
[ ] Sufficient licenses for all target devices
[ ] Licenses assigned in Azure AD
[ ] License activation verified

NETWORK:
[ ] Firewall rules allow MDE URLs (port 443):
    - *.securitycenter.windows.com
    - *.wdcp.microsoft.com
    - *.wd.microsoft.com
    - *.blob.core.windows.net
[ ] Proxy configured if required
[ ] SSL inspection bypass for MDE URLs
[ ] Connectivity tested from each network segment

INFRASTRUCTURE:
[ ] Management tool ready (Intune/GPO/SCCM)
[ ] Onboarding packages downloaded from portal
[ ] Device groups created in portal
[ ] RBAC roles assigned to security team
[ ] Exclusion list prepared

ORGANIZATION:
[ ] Stakeholder approval obtained
[ ] Project timeline agreed
[ ] Support team trained
[ ] Communication sent to users
[ ] Success criteria defined
[ ] Rollback plan documented

TECHNICAL:
[ ] Test devices identified
[ ] Baseline performance captured
[ ] Existing AV removal plan (if applicable)
[ ] Third-party AV coexistence tested

Best Practices
  • Never skip pilot: Always validate with small group first
  • Test connectivity early: Network issues are #1 deployment blocker
  • Document exclusions: Track what and why for future reference
  • Communicate proactively: Let users know what's happening
  • Have rollback plan: Know how to offboard if needed
