Day-to-Day Tasks

Daily Workflow 
TYPICAL DAILY WORKFLOW:

MORNING REVIEW (15-30 min):
1. Check Home dashboard for overnight activity
2. Review new high/critical incidents
3. Check device health status
4. Review threat analytics for new campaigns
5. Check action center for pending items

INCIDENT TRIAGE:
1. Go to Incidents queue
2. Filter: Status = New, Severity = High/Critical
3. Open each incident
4. Review alert story and evidence
5. Assign to analyst or self
6. Begin investigation or escalate

INVESTIGATION:
1. From incident, click affected device
2. Review device timeline
3. Use Advanced Hunting for deep dive
4. Collect investigation package if needed
5. Take response actions
6. Document findings

END OF DAY:
1. Update incident status/notes
2. Check for new critical alerts
3. Review action center completion
4. Handoff to next shift (if 24x7)
5. Log any outstanding items
