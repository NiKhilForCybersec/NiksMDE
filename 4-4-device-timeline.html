Filtering the Timeline

Event Type Filters

Common Filter Combinations 
# Investigation: What did this process do?
Event Types: Process, File, Network, Registry
Search: [process name or PID]

# Investigation: Network activity
Event Types: Network connections, DNS queries
Time: Narrow to incident window

# Investigation: Persistence
Event Types: Registry, File, Scheduled tasks
Search: "Run", "RunOnce", "schtasks"

# Investigation: Credential access
Event Types: Process, Logon
Search: "lsass", "mimikatz", "procdump"

# Investigation: Full attack chain
Event Types: All
Flags: Alerts only (initial filter), then expand

Pro Tips for Timeline Analysis

Timeline Investigation Tips
  • Work from the alert backward: Find root cause before assessing impact
  • Note timestamps precisely: Helps correlate across devices
  • Follow the parent chain: Click parent process to trace execution origin
  • Flag suspicious events: Use the flag feature to mark important findings
  • Export to hunting: Click "Hunt" to pivot to Advanced Hunting for deeper analysis
  • Check all child processes: Malware often spawns multiple processes

  • Reconstruct attack chains from timeline events
  • Identify IOCs for blocking (IPs, hashes, domains)
  • Document findings with timestamps for reports
  • Correlate activity across multiple device timelines
