Best Practices
  • Document actions: Always add comments explaining why
  • Collect evidence first: Get investigation package before remediation
  • Start with containment: Isolate before investigating if active threat
  • Use appropriate isolation: Full for active threats, Selective for investigation

ğŸ’¡ Key Interview Points

  • Device actions: Isolate, restrict apps, scan, collect package, Live Response
  • File actions: Stop/quarantine, add indicator, deep analysis
  • Isolation types: Full (all network) vs Selective (allows O365)
  • Action Center: Track pending and completed actions
  • Permissions: RBAC controls who can execute actions
