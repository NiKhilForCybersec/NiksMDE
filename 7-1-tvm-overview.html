Key TVM Metrics

Exposure Score

The Exposure Score (0-100) reflects your organization's overall vulnerability to threats. Lower is better.

Score Range Risk Level Interpretation
0-29 Low Good security posture, few critical vulnerabilities
30-69 Medium Moderate risk, prioritize high-impact items
70-100 High Significant exposure, immediate action required

Software Inventory

TVM maintains a real-time inventory of all software installed across your endpoints:

KQL - Software Inventory Queries 
// All software with known vulnerabilities
DeviceTvmSoftwareInventory
| where VulnerabilitySeverityLevel != ""
| summarize 
    DeviceCount = dcount(DeviceId),
    Vulnerabilities = count()
    by SoftwareName, SoftwareVersion
| order by DeviceCount desc

// Find devices with specific vulnerable software
DeviceTvmSoftwareInventory
| where SoftwareName =~ "Google Chrome"
| where SoftwareVersion startswith "119"
| project DeviceName, SoftwareName, SoftwareVersion, EndOfSupportStatus

// End-of-life software in environment
DeviceTvmSoftwareInventory
| where EndOfSupportStatus == "EOS Version" or EndOfSupportStatus == "Upcoming EOS"
| summarize DeviceCount = dcount(DeviceId) by SoftwareName, SoftwareVersion, EndOfSupportStatus
| order by DeviceCount desc

Vulnerability Assessment

KQL - Vulnerability Queries 
// Critical vulnerabilities with exploits available
DeviceTvmSoftwareVulnerabilities
| where VulnerabilitySeverityLevel == "Critical"
| where ExploitIsPubliclyAvailable == true
| summarize 
    AffectedDevices = dcount(DeviceId),
    Software = make_set(SoftwareName)
    by CveId, VulnerabilitySeverityLevel
| order by AffectedDevices desc

// Devices exposed to specific CVE
DeviceTvmSoftwareVulnerabilities
| where CveId == "CVE-2024-1234"
| project DeviceName, SoftwareName, SoftwareVersion, RecommendedSecurityUpdate

// Vulnerability trend over time
DeviceTvmSoftwareVulnerabilities
| summarize VulnCount = dcount(CveId) by bin(Timestamp, 1d), VulnerabilitySeverityLevel
| render timechart
