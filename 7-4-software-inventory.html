Advanced Hunting - Software

KQL - Software Queries 
// Find all devices with specific software
DeviceTvmSoftwareInventory
| where SoftwareName has "Chrome"
| summarize Devices = dcount(DeviceName) by SoftwareName, SoftwareVersion
| order by Devices desc

// Find outdated software versions
DeviceTvmSoftwareInventory
| where SoftwareName has "Java"
| where SoftwareVersion !startswith "17" // Not latest major
| project DeviceName, SoftwareName, SoftwareVersion

// Software with known vulnerabilities
DeviceTvmSoftwareVulnerabilities
| where VulnerabilitySeverityLevel == "Critical"
| summarize Devices = dcount(DeviceName), 
            CVEs = dcount(CveId) by SoftwareName
| order by Devices desc

// End of support software
DeviceTvmSoftwareInventory
| where EndOfSupportStatus == "EOS Version"
| summarize Devices = dcount(DeviceName) by SoftwareName, SoftwareVersion
| order by Devices desc

// Browser version distribution
DeviceTvmSoftwareInventory
| where SoftwareName in ("google_chrome", "microsoft_edge", "firefox")
| summarize Count = count() by SoftwareName, SoftwareVersion
| order by SoftwareName, SoftwareVersion desc

API Access

Software Inventory API 
SOFTWARE INVENTORY API:

LIST ALL SOFTWARE:
GET /api/Software
Response: List of all software in organization

GET SPECIFIC SOFTWARE:
GET /api/Software/{softwareId}
Response: Details for specific software

GET SOFTWARE BY DEVICE:
GET /api/machines/{deviceId}/software
Response: Software installed on device

GET VULNERABLE SOFTWARE:
GET /api/Software?$filter=vulnerabilitiesCount gt 0
Response: Software with known CVEs

EXAMPLE RESPONSE:
{
  "id": "microsoft-_-edge",
  "name": "Edge",
  "vendor": "Microsoft",
  "weaknesses": 5,
  "publicExploit": true,
  "activeAlert": false,
  "exposedMachines": 1250,
  "impactScore": 7.8
}

EXPORT TO CSV:
# PowerShell example
$token = Get-MdeToken
$headers = @{ Authorization = "Bearer $token" }
$software = Invoke-RestMethod -Uri "https://api.securitycenter.microsoft.com/api/Software" `
    -Headers $headers
$software.value | Export-Csv -Path "software_inventory.csv"
