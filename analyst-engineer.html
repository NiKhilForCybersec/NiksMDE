Tier 1 - Alert Triage

  • Review overnight alert queue in M365 Defender portal
  • Triage high/critical severity alerts first
  • Determine: True positive, false positive, or benign?
  • Quick response: Isolate device, stop process, quarantine file
  • Escalate complex cases to Tier 2

Tier 2 - Investigation & Response

  • Deep investigation: Full attack chain reconstruction
  • Advanced Hunting: KQL queries for threat hunting
  • Live Response: Remote forensics and remediation
  • Incident Management: Coordinate response activities
  • IOC Extraction: Identify and submit indicators

Tier 3 - Threat Hunting & Forensics

  • Proactive threat hunting: Hypothesis-driven searches
  • Malware analysis: Reverse engineering, sandbox analysis
  • Digital forensics: Memory dumps, disk imaging
  • Detection development: Work with engineers on new rules
  • Purple teaming: Validate detections with red team
