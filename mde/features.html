Microsoft Defender Antivirus

Microsoft Defender Antivirus is the built-in next-generation protection engine providing real-time, behavior-based, and heuristic antivirus protection. It's tightly integrated with MDE for enhanced cloud-delivered protection.

Protection Capabilities

Feature Description Configuration
Real-Time Protection On-access scanning of files, processes, and registry Intune, GPO, PowerShell
Cloud-Delivered Protection Cloud-based ML analysis for unknown files (Block at First Sight) Requires internet, configurable timeout
Behavior Monitoring Detects malicious behavior patterns in processes Always enabled with real-time
Tamper Protection Prevents unauthorized changes to security settings Portal toggle, Intune policy
PUA Protection Blocks potentially unwanted applications Audit or Block mode
Scheduled Scans Quick, Full, or Custom scans on schedule GPO, Intune, PowerShell

PowerShell Configuration

PowerShell 
# Check Defender Status
Get-MpComputerStatus | Select-Object AntivirusEnabled, RealTimeProtectionEnabled, 
    IoavProtectionEnabled, AntispywareEnabled, BehaviorMonitorEnabled

# Enable Cloud-Delivered Protection
Set-MpPreference -MAPSReporting Advanced
Set-MpPreference -SubmitSamplesConsent SendAllSamples

# Configure Block at First Sight
Set-MpPreference -DisableBlockAtFirstSeen $false

# Enable PUA Protection
Set-MpPreference -PUAProtection Enabled  # 1=Enabled, 2=Audit, 0=Disabled

# Add Exclusions (use sparingly!)
Add-MpPreference -ExclusionPath "C:\CustomApp"
Add-MpPreference -ExclusionProcess "customapp.exe"
Add-MpPreference -ExclusionExtension ".customext"

# View Current Exclusions
Get-MpPreference | Select-Object -ExpandProperty ExclusionPath

# Run Quick Scan
Start-MpScan -ScanType QuickScan

# Update Definitions
Update-MpSignature

Exclusion Warning

Exclusions create blind spots in protection. Always validate exclusion requests and prefer process/path exclusions over extension exclusions. Document all exclusions with business justification.
